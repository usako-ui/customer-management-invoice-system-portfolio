## 🎯 システム概要

### これ一つで完結する顧客管理＆請求書発行

**Googleアカウントだけで導入できる**、個人事業主・小規模事業者向けの統合管理システムです。

従来スプレッドシートやExcelで手動管理していた業務を自動化し、**作業時間を70〜80%削減**します。

### なぜこのシステムを作ったのか

私自身が個人事業主として活動する中で感じていた課題:

- 「この担当者さん、どこの会社だっけ?」
- 「株式会社〇〇? 〇〇株式会社? どっちだっけ?」
- スプレッドシートの小さなマスで見づらい、探しづらい
- 請求書を毎回手作業で作って、保存して、メール添付して...

**前職で使っていた企業の顧客管理システムのような便利さを、個人事業主でも実現したい**という思いから開発しました。

## 💡 こんな課題を解決

### 従来の困りごと

- ❌ スプレッドシートで顧客を探すのに時間がかかる
- ❌ 請求書作成→PDF化→保存→メール送信が毎回面倒
- ❌ 顧客数が増えると管理が煩雑になる
- ❌ 有料の業務管理ソフトは導入コストが高い

### このシステムなら

- ✅ **検索機能**で顧客情報をすぐに見つけられる
- ✅ **ワンクリック**で請求書PDF作成＆自動保存
- ✅ **Gmail下書き**も自動作成で送信がスムーズ
- ✅ **完全無料**でGoogleアカウントだけで導入可能

### こんな方におすすめ

- **スプレッドシート・Excelで顧客管理している方**
  - 顧客数が増えて管理が煩雑になってきた
  - 顧客情報を探すのに時間がかかる

- **毎月複数の請求書を発行している方**
  - 請求書作成の繰り返し作業が面倒
  - ファイル管理が煩雑

- **業務効率化したいけど何から始めればいいか分からない方**
  - システム導入にコストをかけられない
  - 自動化の方法が分からない

### 向いている業種

- 🎬 動画編集者・クリエイター
- 💻 Webデザイナー・エンジニア
- 📊 コンサルタント
- 🏛 士業（税理士、行政書士など）
- 🏪 その他BtoB取引のある個人・小規模事業者

## ✨ 主な機能

### 1. 顧客管理機能

| 機能 | 説明 |
|------|------|
| **顧客情報登録** | 顧客名、担当者、連絡先、住所を一元管理 |
| **顧客ID自動採番** | C001形式で自動採番（手動入力不要） |
| **検索機能** | 企業名・担当者名・電話番号で部分一致検索 |
| **顧客詳細表示** | 見やすい画面で情報確認・編集 |
| **情報更新・削除** | 顧客情報の編集・削除が簡単操作 |

### 2. 請求書発行機能

| 機能 | 説明 |
|------|------|
| **請求書自動作成** | テンプレートから請求書を自動生成 |
| **請求書番号自動採番** | INV-2026-01-001形式で自動採番 |
| **PDF自動生成** | スプレッドシートから自動でPDF化 |
| **フォルダ自動保存** | 顧客別フォルダを自動作成して保存 |
| **Gmail下書き作成** | 請求書PDFを添付したメール下書きを自動作成（任意） |
| **請求書一覧管理** | 発行済み請求書の一覧表示・検索 |
| **請求書再発行** | 修正時の再発行機能（元の請求書は取消状態に） |
| **支払状況管理** | 未振込・振込済の状態管理 |

### 3. セキュリティ機能

- **パスワード認証** - システムアクセス時の認証
- **セッション管理** - ログイン状態の管理

## 🌟 導入のメリット

### 作業時間を大幅削減

#### 従来の作業（手作業）

```
1. スプレッドシートで顧客情報を探す
2. 請求書テンプレートを開く
3. 顧客情報を手入力（コピペ）
4. 明細を入力
5. 金額計算
6. PDF化
7. ファイル名を変更
8. フォルダを探して保存
9. Gmailを開く
10. 宛先、件名、本文を入力
11. PDFを添付
12. 送信 or 下書き保存

所要時間：約5〜10分/件
```

#### このシステムでの作業（自動化）

```
1. 顧客一覧から「請求書作成」をクリック
2. 明細を入力
3. 「請求書を作成」をクリック
   ↓
【以下すべて自動処理】
・請求書番号の自動採番
・テンプレートへの自動反映
・PDF化
・顧客別フォルダの自動作成
・フォルダへの自動保存
・Gmail下書き作成（任意）

所要時間：約1〜2分/件
```

**→ 約70〜80%の時間削減を実現**

### システムの特徴・工夫

#### 1. ユーザビリティ重視

- 画面付きシステムで**入力ミスを防止**
- スプレッドシートより**見やすく、入力しやすい**UI設計
- 部分一致検索で**素早く顧客を特定**

#### 2. 柔軟な運用設計

- Gmail下書き作成は**任意選択式**
  - ユーザーによって業務フローが異なるため
  - メールで送る人、PDFを別の方法で送る人など、様々な運用に対応

#### 3. データの正確性確保

- 顧客ID・請求書番号の**自動採番**で入力ミス防止
- 請求書再発行時は**元の請求書を取消状態**にして履歴を保持
- 金額計算の**自動化**で計算ミス防止

#### 4. 拡張性の考慮

- 顧客情報の項目は**必要最低限**に設定（将来的に追加可能）
- テンプレートは**カスタマイズ可能**（業種や好みに応じて変更可能）

### なぜGoogle Apps Scriptを選んだのか

1. **導入コストゼロ** - Googleアカウントがあれば誰でも無料で使える
2. **運用が簡単** - サーバー管理不要、メンテナンスフリー

4. **小規模事業者に最適** - 初期コストをかけずに本格的なシステムが構築できる
